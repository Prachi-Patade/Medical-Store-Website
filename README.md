HealthCare Plus – Online Medical Store

Project Overview
HealthCare Plus is a comprehensive online medical store platform designed to provide users a seamless shopping experience for medicines, wellness products, baby care, beauty, and elderly care essentials. 
The website focuses on user-friendly navigation, product browsing, and cart management while maintaining a clean and professional UI.

🛠Technologies Used: HTML, CSS, JavaScript

✨Key Features: Responsive design, dynamic cart, product galleries, search functionality, user authentication

📁Project Type: Academic / Web Development Project

✨Website Pages & Features
1. Home Page (main.html)
-Displays main logo, search bar, cart icon, and sign-in link
-Horizontal navigation bar for easy category selection
-Product Galleries:
-Wellness & Immunity Care
-Fitness & Nutrition
-Ayurveda & Herbal Products
-Product Cards: Image, name, price, rating stars, dual buttons (Buy Now & Add to Cart)
2. Medicine Page (medicine.html)
-Shows medicines with images, prices, ratings
-Dual buttons for adding to cart or direct purchase
3. Baby Care Page (babycare.html)
-Products for infants: formula, diapers, wipes, skincare
4. Beauty Page (beauty.html)
-Cosmetic and skincare products
-Responsive product card layout
5. Elderly Care Page (elderlycare.html)
-Products for senior citizens: supplements, mobility aids, hygiene essentials
6. Health Page (health.html)
-Health corner products: immunity boosters, vitamins, hygiene essentials
7. Terms & Conditions Page (terms.html)
-Legal and privacy information
-Guidelines for users and website usage
8. Cart Page (cart.html)
-Dynamic cart showing added products
-Quantity adjustments and total price calculation
9. Authentication Flow
-Sign Up Page (signup.html) – New user registration
-Login Page (login.html) – Existing user login
-Forgot Password Page (forgot.html) – Password reset

Key Functionalities
1. Dynamic Cart Management
-Users can add products to cart from any page
-Quantity automatically updated for duplicate products
Cart data stored in LocalStorage for persistent shopping session
2. Location-Based Delivery Info
-Users can provide delivery address via URL query or select manually
-Stored in LocalStorage for continuity across pages
3. Responsive Design
-Optimized for both desktop and mobile
-Flexbox-based product galleries for smooth alignment
4. Product Ratings
-Star ratings displayed on each product
-Visual appeal with color-coded star icons

✨Conclusion
HealthCare Plus is an interactive and professional medical store website, demonstrating:
Product catalog management
Dynamic cart functionality
User-friendly interface
Responsive and visually appealing design
This project is ideal for showcasing web development skills using HTML, CSS, and JavaScript.
